# MD Location Recognition Portal (Live Map + Drawing)

Adds **Leaflet.draw** tools to capture site geometry on the live map, with:
- Polygon/rectangle/polyline/marker drawing
- Edit/delete existing shapes
- Automatic **area/perimeter** measurement (approx geodesic)
- **Export** to GeoJSON and **Import** from GeoJSON
- Shapes persist in **localStorage**

Sample points included:
- Bangsar — 3.1290, 101.6840 — *MD Hub*
- Plaza Azalea, Shah Alam — 3.0738, 101.5183 — *MD Nexus*
- Iskandar Puteri — 1.4190, 103.6550 — *MD Tech Zone*

## How to run
1. Download and unzip.
2. Open `index.html` (Internet required to load Leaflet and OSM tiles).
3. Sign in, go to **Map**, click **Start Drawing** (or use the toolbar on the map).
4. Export shapes as `mdloc-shapes.geojson`, or import your own GeoJSON.
